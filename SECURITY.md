# Security Policy / 安全策略

Please do not report security issues in public GitHub issues.

For now, report privately to the project maintainer.

请不要在公开 GitHub Issue 中报告安全问题。

当前阶段请私下联系项目维护者。

## Sensitive Data Rules / 敏感数据规则

Do not log:

不要记录以下内容：

- Passwords
- Password hashes
- Refresh token plaintext
- JWT private keys
- Email verification codes
- Full Authorization headers
- 密码
- 密码哈希
- refresh token 明文
- JWT 私钥
- 邮箱验证码
- 完整 Authorization 请求头
